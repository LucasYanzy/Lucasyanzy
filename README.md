## 🚀 Key Projects | 核心项目

### [Magic Glove (IEEE Build-a-thon Winner)](https://github.com/LucasYanzy/Magic-Glove)
*   **Description**: A wearable hardware controller developed using C++ and Arduino IDE. It translates natural hand movements into digital commands, replacing traditional input devices.
*   **Awards**: Winner of Rutgers IEEE "Best Productivity / Automation Challenge" Prize.
*   **描述**: 利用 C++ 及 Arduino IDE 开发的可穿戴硬件控制器。将自然手势转化为数字指令，旨在替代传统输入设备。
*   **奖顶**: 荣获 Rutgers IEEE “最佳生产力与自动化挑战”奖项。

### [Interactive Web Portfolio](https://github.com/LucasYanzy/Intro)
*   **Description**: A high-performance, single-page professional portfolio featuring a custom 30fps particle nebula background, parallax effects, and bilingual content management.
*   **Tech Stack**: HTML5, CSS3 (Glassmorphism), JavaScript (Canvas API).
*   **描述**: 高性能、单页专业简历，具有自定义 30fps 粒子星云背景、视差效果和中英双语内容。
*   **技术栈**: HTML5, CSS3, JavaScript (Canvas API).
